Border Queue App

The Border Queue Reservation System is a full-stack application designed to streamline border crossing by enabling users to book time slots in advance. It ensures efficient queue management, generates confirmation PDFs with QR codes, and provides border officials with verification tools.

  • Reservation System: Users can view available slots and book their crossing time.
  • PDF Ticket Generation: Confirmation includes a PDF with personal details and QR code.
  • Authentication: Secure login and registration powered by Supabase.
  • Admin Tools: Guards can validate tickets and manage capacity.
  • Responsive Design: Optimized for both desktop and mobile use.

Casino Lobby

The Casino Lobby is a modern frontend project simulating a real-world online casino interface. It focuses on clean UX, interactive components, and engaging visuals.

  • Game Showcase: Games are displayed by category (Roulette, Slots, Blackjack, etc.) in stylish cards.

  • Video UI Elements: Category cards trigger dynamic videos on hover for enhanced immersion.
